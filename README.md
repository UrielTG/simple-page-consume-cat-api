
# Catlify

Catlify is a simple webpage in which an API is consumed in order to show random cats images.
The API is 'Cat as a service (CATAAS)'

## API Reference
This API is easy to use:
#### API LINK
```
https://cataas.com/#/
```

#### Get item

```
  https://cataas.com/cat
```

| URL         | Description               |
| :---------- | :------------------------ |
| `/cat`      | Will return a random cat  |
